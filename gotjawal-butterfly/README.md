# 🦋 곶자왈 나비를 피해라 — 돌하르방 대모험

어두운 곶자왈 숲에서 **돌하르방**이 되어, 사방에서 날아오는 **나비를 피하는** 단일 HTML 게임(Canvas).
중간에 나타나는 **새**를 잡으면 곶자왈 나비·새 퀴즈가 나오고, 맞히면 새가 친구가 되어 **보너스 점수**.
새 친구 **5마리**를 모으면 귀한 **제주고사리삼**이 나타나고, 거기에 닿으면 **대성공**으로 큰 점수를 얻고 끝납니다.

## 조작
- ⌨️ 키보드 **방향키(← ↑ → ↓)** 또는 **WASD** 로 돌하르방을 움직여요. (PC 기본 조작)
- 📱 휴대폰에서는 키보드가 없으니 **손가락으로 끌어서** 움직여요.
- 🦋 나비에 **닿으면** 생명 ❤️ −1 (피격 직후 잠깐 무적). 나비는 시간이 갈수록 **점점 많고 빨라져요**.
- 🐦 새에 닿으면 **퀴즈**가 나와요(이때 게임은 잠깐 멈춤). 맞히면 새 친구 +1, **+500점**.
- 🌿 새 친구 5마리 → **제주고사리삼** 등장 → 닿으면 승리(**+2000점 & 남은 생명 보너스**).

## 게임 설계 메모 (요청 반영)
| 요청 | 구현 |
|---|---|
| 어두운 숲에서 나비가 날아옴 | 깊은 곶자왈 숲 배경 + 반딧불이, 가장자리 사방에서 나비 스폰 |
| 캐릭터 = 돌하르방 | 현무암 질감·벙거지 모자·큰 코의 돌하르방을 Canvas로 그림 |
| 나비가 점점 많아짐 | 스폰 간격이 시간에 따라 짧아지고(1500ms→360ms), 속도도 증가 |
| 새 = 먹으면 퀴즈 | 새와 충돌 시 퀴즈 오버레이, 게임 일시정지 |
| 맞히면 새 획득 + 추가 점수 | 정답 시 새 친구 +1, +500점 |
| 마지막 제주고사리삼 = 끝 + 큰 점수 | 새 5마리 모으면 제주고사리삼 등장, 닿으면 승리 + 보너스 |
| 퀴즈는 QUIZ.md 2단계(나비·새) | `gotjawal-quiz/QUIZ.md` 2단계 10문제를 그대로 사용 |

## 바로 해보기 (로컬)
`index.html` 파일을 더블클릭해서 브라우저로 열면 끝. 휴대폰도 OK.

## 배포 (GitHub Pages)
`.github/workflows/deploy-game.yml` 가 `games/` 폴더를 Pages로 자동 배포한다.
- main에 `games/` 변경이 push되면 자동 배포 (수동: Actions → *Deploy game to GitHub Pages* → Run)
- 배포 주소: **https://dream0grow.github.io/dream-grow-content-automation/gotjawal-butterfly/**

## 문제 추가·수정
`index.html` 의 `QUIZ` 배열에서 `{ q:"질문", a:"정답", w:["오답1","오답2"] }` 형식으로 더하거나 고치면 된다.
내용은 `gotjawal-quiz/QUIZ.md` 2단계와 동일하게 맞춰두면 관리가 편하다.
