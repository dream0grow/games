# 🪲 곶자왈 곤충 찾기 타워

아이 컨셉으로 만든 **단일 HTML 파일 게임**(Canvas). 라이브러리·빌드 없음 → 브라우저에서 바로 열림, PC·휴대폰 모두 동작.

## 바로 해보기
`gotjawal-tower.html` 파일을 더블클릭(또는 브라우저로 열기)하면 끝.

## 게임 규칙
- 🪲 **물장군**이 되어 타워를 점프해서 올라간다 (로블록스 타워 점프 게임 벤치마킹).
- 💣 **폭탄은 하늘에서 떨어진다.** 닿으면 즉시 아웃! (떨어질 자리에 ⚠️ 경고 표시) **높이 올라갈수록 폭탄이 더 자주·더 많이** 떨어짐.
- 🌋 **용암이 빠르게 차오른다(3배 속도)** — 절대 멈추지 말 것. 오른쪽 세로 게이지로 위험도 확인.
- 📟 **계기판**: 현재 높이(m)와 점수를 실시간 표시.
- 🧱 **금 간 블록**은 밟으면 사라짐.
- ❓ **물음표 블록**에서 곤충 퀴즈 — **랜덤 출제**(순서 고정 아님, 미수집 곤충 우선).
- 🐾 정답 시 **곤충 카드(실제 사진 + 설명)**가 떠서 **슬롯으로 날아가 모이는 애니메이션**.
- 🪲 곤충 **20종 도감**. **수집할 때마다 +500점** — 다 모아도 끝나지 않고 **끝없이** 점수 도전.
- ❤️ 목숨 5개. 죽어도 목숨이 있으면 **죽은 그 자리에서 이어서** 진행(주변 폭탄 제거+잠깐 무적).
- 🏁 목숨을 다 쓰면 **END** 화면 — 점수와 **랭킹(이 기기 최고기록)** 표시.

## 동물 카드 사진 넣기
`games/assets/` 폴더에 정해진 파일명으로 사진을 넣으면 카드/슬롯에 바로 반영됩니다.
넣기 전에는 코드의 공개 URL→자동 일러스트 순으로 대체되어 항상 정상 동작. 자세한 건 `assets/README.md` 참고.

## 조작
- PC: `←` `→` 이동, `Space`/`↑` 점프
- 폰: 화면 아래 터치 버튼(◀ ▶ 점프)

## 배포 (공유 URL 만들기)
단일 파일이라 어디든 올리면 끝:
- **GitHub Pages**: repo Settings → Pages → 이 폴더 게시 → `.../games/gotjawal-tower.html`
- **Netlify / Cloudflare Pages**: 파일 드래그&드롭 한 번이면 URL 발급
- 발급된 URL로 QR코드를 만들면 아이가 친구·가족과 바로 공유 가능

## 검증
게임 로직은 Node 환경에서 자동 테스트로 확인:
- 메커닉 단위 테스트 18종 통과(랜덤 퀴즈덱/미수집 우선/카드 수집/슬롯 채움/CLEAR/떨어지는 폭탄 충돌/같은 자리 이어하기/폭탄 제거·무적/고도별 폭탄 증가/용암 이어하기/END·랭킹/피한 폭탄 카운트)
- 렌더 경로 무오류(수천 프레임 연속 렌더, 폭탄·경고·게이지·카드 레이어 포함), 봇 플레이로 난이도 균형 확인
