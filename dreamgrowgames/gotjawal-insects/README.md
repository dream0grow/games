# 곶자왈 곤충 탐험대 🐛 (스테이지 1)

초등 고학년(5~6학년)까지 즐기는 **운 + 추리** 곤충 찾기 게임. 단일 HTML 파일.

## 플레이 방법
1. `assets/` 폴더에 사진을 넣습니다 (→ `assets/README.md` 참고).
2. `index.html`을 브라우저로 엽니다 (휴대폰도 OK).
3. **똑같이 생긴 돌**을 눌러 안에 뭐가 있는지 확인해요.

## 게임 규칙 (핵심: 지뢰찾기식 추리)
- 🪨 **돌 16개는 전부 똑같이 생겼어요.** 눌러야 내용을 알 수 있어요 → **운**.
- 🔢 안전한 돌을 까면 **숫자**가 떠요 = **둘레 8칸에 숨은 숨골 개수**. 숫자로 위험을 **추리** → **스킬**.
- 🕳️ **숨골**(돌 5개에 숨음)을 누르면 목숨 −1. ❤️ 하트를 찾으면 목숨 +1.
- 🌳 **나무**는 겉으로 구분 안 돼요. 누르면 퀴즈(맞히면 🔦돋보기로 숨골 한 곳을 🚩 표시) — 단 **썩은 나무**면 쓰러져 목숨 −1.
- 🐛 곤충을 찾으면 **그 곤충 퀴즈**를 풀어요. 맞히면 보너스 ⭐.
- 목숨 3개로 시작, 곤충 3마리를 다 찾으면 승리. 점수 = 곤충 + 퀴즈 + 남은목숨 + 시간보너스.

## 수정 반영 (v2)
1. **배경을 실제 사진으로** (`assets/배경.jpg`).
2. **곤충 사진을 로컬 파일로**, 파일명에 곤충 이름 (`assets/왕사마귀.jpg` 등).
3. **단서를 없애 운+추리 게임으로** — 돌·나무가 똑같이 생겨 눌러봐야 알고, 숫자로 숨골을 추리.
4. **채집 후 곤충 퀴즈** 풀기.

## 사진이 없을 때
사진 파일이 아직 없으면 곤충 이모지로 임시 표시되어 **게임은 그대로 작동**합니다.
사진을 `assets/`에 넣는 즉시 진짜 사진으로 바뀝니다.
